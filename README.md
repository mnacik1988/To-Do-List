# To-Do-List

**Mynado** — умный To-Do список, который понимает голос и думает вместе с тобой.

Просто скажи, что нужно сделать — Mynado сам расставит дату, время и приоритет, а если что-то поменялось, поправит задачу голосом или текстом за секунды. Никаких лишних тапов и полей для заполнения: встроенный ИИ-помощник (на базе Claude) берёт рутину на себя, а push-напоминания не дадут забыть о важном, даже если приложение закрыто.

- 🎙️ **Голосовой ввод** — надиктуй задачу, дату и время естественной речью
- 🤖 **ИИ-помощник** — понимает контекст, сам расставляет приоритеты и правит задачи по команде
- 🔔 **Push-напоминания** — приходят вовремя, даже офлайн
- 🌍 **5 языков интерфейса** — RU, EN, UK, DE, FR
- 📴 **Работает офлайн** — задачи и напоминания доступны без интернета
- 🎨 **Аккуратный дизайн** — тёмная и светлая темы, спокойная navy+gold палитра

Доступно как PWA/TWA-приложение для Android (Google Play) и iOS (сохранение на экран через Safari).
