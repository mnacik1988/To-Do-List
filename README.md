# To-Do-List

**Mynado** — голосовой To-Do список с ИИ-помощником (Claude), push-напоминаниями и офлайн-режимом. PWA/TWA, доступно в Google Play (внутреннее тестирование).

- 🎙️ Голосовой ввод задач
- 🤖 ИИ разбирает и правит задачи (дата, время, приоритет)
- 🔔 Push-уведомления о напоминаниях
- 🌍 5 языков интерфейса (RU, EN, UK, DE, FR)
- 📴 Работает офлайн

Живая версия: https://mnacik1988.github.io/To-Do-List/
